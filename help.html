Welcome to the Run Tracker!

We make tracking your run history easy assuring you stay on top of your goals!

How to View Your Runs

Click on the "View Runs" button from the Home Screen

How to Create a New Run

1. Navigate to the Home Screen by clicking "Home"

2. Navigate to the View Runs screen by clicking "View Runs"

3. Locate the "Add New Run" button and click it

How to Edit an Existing Run

1. Navigate to the Home Screen by clicking "Home"

2. Navigate to the View Runs screen by clicking "View Runs"

3. Select the run you would like to edit, and click on it
